# 🛰 Welcome to the MERI open source community!!

### MERI  👉🏻 [(HU)📜](https://github.com/Fizikova-Csindra/MERI/blob/main/README.hu.md)
#### Magyar Egyetemek Robotika Iskolája/Mechanics Education in Robotics Institut.
This site hosts the various open source robot building projects managed by the Csindra Foundation and its Partners, including:

- development and coordination of educational and training materials for 6-22 year olds,
- support for robot development and robotics research,
- support for the development of information and communication technologies,
- support for scientific research related to space exploration,
- creation and support of own research groups, GitHub operations.

Maintaining these projects and managing community contributions is a difficult task. Please support us with [donations](https://web.csindra.hu) and [purchase of original Csindra products](https://web.csindra.hu/).
### ✨ How you can contribute

- **Solve open problems**: try to reproduce the problems reported by other users and confirm that you are experiencing them, or ask users for more details if necessary. Improve descriptions. Help users who ask for support.
- **Submit fixes and implementations**: choose a help request that you think you can implement yourself and submit a documentation along with an implementation.
- **Test solution** for an open implementation: try to make the suggested changes and describe success or failure. Testing on real hardware takes time, and any help in this will speed up our work together.
-  **Help others** solve their problems and suggest good ways to implement fixes and features.
- **Your documentation** helps improve existing content.

We value contributions, and the most active and consistent contributors are rewarded with public recognition, a certificate, and medals (BATKA, TALENT, KARAT) according to the Csindra Foundation's value system.
